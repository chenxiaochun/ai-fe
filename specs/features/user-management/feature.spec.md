# 功能规格：用户管理

- ID: `user-management`
- 功能名称：`user-management`
- 类型：`crud-page`
- 状态：`learned`
- 路由：`/admin/users`

## 目标

实现一个用户管理页面，支持搜索、分页、新增和编辑用户。

## 范围

### 包含

- 展示用户列表
- 按关键词搜索用户
- 分页浏览用户
- 新增用户
- 编辑用户

### 不包含

- 删除用户
- 批量导入
- 角色分配

## 用户

- 管理员
- 运营人员

## 能力

- 用户列表
- 搜索
- 分页
- 新增用户
- 编辑用户
